Detailed Description

template<typename key_t, typename entry_t>
class stf::ds::indexed_list< key_t, entry_t >

A class that stores a list with an overlaid index for quick access for a key.

Template Parameters
key_tThe type of the key that is used to index the list
entry_tThe type of the entry that is stored in the list
Note
key_t must have a hash function defined that is compatible with std::unordered_map
entry_t must have a correct move constructor
