# support

`stf` officially supports the following combinations of operating systems and compilers.
Other combinations may work but are not actively tested in our automated build.

## windows

* Visual Studio 16
* Visual Studio 17
* g++ 12

## ubuntu

* clang++ (versions 16, 17, 18)
* g++ (versions 12, 13, 14)

## macos

* clang++ 15
* g++ (versions 12, 13, 14)
