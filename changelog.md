# Changelog

All notable changes to this project are tracked in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).
This project does its best to adhere to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).
There is some looseness on ABI changes because it is a header-only library so recompilation is already necessary when changing versions.
